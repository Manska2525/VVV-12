# README

Проект создан в GigaStudio

## Стек

- Next.js 16.2.1
- TypeScript
- Tailwind CSS 3.4.18
- shadcn/ui
- React 19.2.0

## Локальный запуск
```bash
bun install
bun run dev
```

Откройте http://localhost:3000

## Деплой

[Документация Cloud.ru](https://cloud.ru/docs/container-apps-evolution/ug/topics/quickstart)

Важно: в репозитории раньше была дублированная копия проекта в папке `gigastudio-d61998ff`. Теперь единственный источник правок — `src/` в корне репозитория. Удалённые копии не влияют на деплой и не должны добавляться обратно, чтобы избежать конфликтов и устаревшего кода.
